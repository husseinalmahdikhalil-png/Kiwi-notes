Kiwi Notes is a calm, audio-powered language learning app.

It helps users learn vocabulary using their *own personal content*, not predefined courses.

## What makes Kiwi Notes different?
- Add your own words and sentences
- Turn them into natural audio
- Listen like a personal podcast
- Review using test-based spaced repetition
- No streaks, no gamification, no noise
